# Fricoot

**Description** :
Un projet web en HTML/CSS/PHP avec une base de données MySQL.

---

## Fonctionnalités
- Interface utilisateur responsive
- Gestion des données via PHP/MySQL
- Formulaires interactifs

---

## Prérequis
- Serveur web (Apache/Nginx)
- PHP 7.4+
- MySQL 5.7+
- Composer (si dépendances PHP)

---

## Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/Laurent-Lebarbier/fricoot.git
   cd fricoot
   ```

2. Importer la base de données :
   - Créer une base de données MySQL.
   - Importer le fichier `fricoot.sql` (à fournir ou à générer).

3. Configurer la connexion à la base de données :
   - Modifier le fichier `config.php` avec vos identifiants MySQL.

4. Déployer sur un serveur local ou en ligne.

---

## Structure du projet
- `index.php` : Page principale
- `css/` : Feuille de style
- `includes/` : Fichiers PHP inclus
- `assets/` : Images et ressources
